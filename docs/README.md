### 컴포넌트 지연 로딩

```
페이지 코드 자체를 분할하는 대신 단일 컴포넌트를 분할하여 컴포넌트가 쓰이는 순간에 불러오도록 할 것이다.
```

### 컴포넌트 사전 로딩

```
컴포넌트를 분할하여 지연 로딩을 적용하면, 첫 화면 진입 시 분할된 코드 중 당장 필요한 코드만 다운로드 하기 때문에 첫 화면을 더 빠르게 그릴 수 있게 된다.
하지만 서비스 이용 과정에서 분할된 컴포넌트를 사용하려고 할 떄, 다운로드되어 있지 않은 코드를 추가로 다운로드하는 시간만큼 서비스 이용에 지연이 발생한다.

이런 문제를 해결하기 위해서, 코드를 분할하여 첫 화면 진입 시에는 다운로드하지 않지만, 이후 해당 코드가 필요한 시점보다는 먼저 코드를 로드하여 해당 코드를 지연 없이 사용할 수 있도록 하는 컴포넌트 사전 로딩 기법을 살펴볼 것이다.
```

## 이미지 사전 로딩

```
이미지도 컴포넌트와 마찬가지이다.
이미지를 필요한 시점에 로드하면 이미지가 다운로드 하는 시간만큼 기다려야 한다.
그래서 이미지도 필요한 시점보다 먼저 다운로드해 두고, 필요할 때 바로 이미지를 보여줄 수 있도록 하는 이미지 사전 로딩 기법을 적용해 볼 것이다.
```
